## System description

DBNOD is an R and Java implementation of a complete multivariate time series (MTS) outlier detection system covering problems from pre-processing to post-score analysis. It adopts dynamic Bayesian network (DBN) modeling with conjunction of a sliding window approach to uncover observations suspected of not have been generated by data’s underlying processes.
